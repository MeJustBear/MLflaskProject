### Сборка
Клонируем репозиторий, собираем образ с помощью compose.

```
$ git clone https://github.com/MeJustBear/ml-flask-test-task
$ cd cd ml-flask-test-task/
$ docker-compose up
```

### Запуск

```
 $ docker run --name my-container -d -p 8080:8080 ml-flask-test-task
```
